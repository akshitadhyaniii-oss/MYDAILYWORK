Titanic Survival Prediction

Objective
To predict whether a passenger survived the Titanic disaster using machine learning techniques.

Dataset
- Training data: Titanic dataset (`train.csv`)
- Testing data: Provided Kaggle test dataset

Model Used
Logistic Regression

Accuracy
81%

Workflow
1. Data Cleaning
2. Feature Encoding
3. Train-Test Split
4. Model Training
5. Model Evaluation

Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
